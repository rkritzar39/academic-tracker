# Rocket Academic Tracker

A responsive, browser-based academic tracker prototype built with HTML, CSS, and JavaScript. 

## Run locally

Open `index.html` in a browser.

## Publish with GitHub Pages

1. Upload all files and folders to a GitHub repository.
2. Open **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select **main** and **/(root)**, then save.

## Data and privacy

Tracker information is saved in the browser with `localStorage`. Do not commit real grades, student IDs, passwords, access tokens, or private student records to a public repository. Exported JSON backups should also be treated as private.

## Branding note

This is an unofficial student prototype. It uses a generic rocket-inspired visual identity and does not include official University of Toledo logos.


## V2 improvements

V2 adds a restrained liquid-glass presentation, atmospheric backgrounds, per-course accent colors, improved hierarchy, responsive mobile bottom navigation, enhanced focus styles, reduced-motion support, polished dialogs, and more refined interaction states.


## V3 Advanced

Adds study-session logging, degree requirement tracking, analytics, risk alerts, priority summaries, workload visualization, course-performance bars, and additional local-data insights.


## V4 Calendar

Adds a unified monthly calendar for assignments, exams, official UToledo 2026–2027 academic dates, and personal campus events. Includes filtering, date details, upcoming events, month navigation, and iCalendar export. Official dates are sourced from the UToledo Office of the Provost and should be rechecked for changes.


## V5 Recurrence, archived semesters, and profile

Adds daily, weekday, weekly, biweekly, and monthly personal-event recurrence; selectable semester calendar views from Fall 2025 through Summer 2027; and a locally stored, resized profile picture plus editable display name, school, and program. Recurring events are expanded in the calendar and iCalendar export.
